# Weather Dashboard

## Description

A Javascript-powered Weather Dashboard that displays your 5 day forcast for a searched area.

## Installation & Access

To access the project, visit my [GitHub Pages](https://patrickbrown-io.github.io/weather-dashboard-application/)
